Requisitos de sistema:  
    - GHC versión 9.4.7 (o posterior)

Ejecución:  
    Abrir terminal dentro de la carpeta raíz  
    Ejecutar Make  
    Iniciar el juego con el comando:  
        $ ./max n s  
    Donde n es el tamaño del tablero (n*n) y s es la semilla
